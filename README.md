# Tic-Tac-Toe AI (Unbeatable)

A desktop application written in Python that implements the classic Tic-Tac-Toe game. This project features an unbeatable AI opponent powered by the **Minimax algorithm**.

## 🧠 Technical Overview
- **Core Logic:** Uses the **Minimax algorithm** to ensure the computer always makes the optimal move, making it impossible to beat.
- **Interface:** Built with **Tkinter** for a clean, responsive desktop user interface.
- **UX:** Includes a programmed 1-second delay for the AI's move to simulate "thinking" and provide a better user experience.

## 🚀 Features
- **Unbeatable AI:** Perfectly optimized moves.
- **Interactive GUI:** Styled buttons and status alerts.
- **OOP Design:** Clean, modular, and maintainable Python code structure.

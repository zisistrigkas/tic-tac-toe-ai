# ❌⭕ Unbeatable Tic-Tac-Toe AI

> *A sophisticated desktop game where you compete against an unbeatable AI opponent.*

## ⚡ Core Engine
- **Minimax Algorithm:** The brain of the operation. It calculates every possible move to ensure the computer never loses.
- **Tkinter GUI:** Clean, modern interface for a smooth user experience.
- **Smart UX:** Includes a 1-second delay for the AI to "think," making the game feel natural.

## 🛠 How to Run
Download: Click the green [Code] button -> [Download ZIP].
Extract: Unzip the files.
Open Terminal: Open PowerShell or Command Prompt in the project folder.
Execute: Run the following command: python triliza.py
